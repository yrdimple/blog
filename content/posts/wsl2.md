# how to install WSL2

## window11 & WSL2

```shell
wsl --install

wsl --install -d Ubuntu-20.04
```

直接就完成了安装。wsl2 安装也太方便了吧！

注:

第一步 会比较快一点

第二步 安装 Ubuntu 可能就是一个漫长的等待了
