# jekyll hexo or hugo ?


- jekyll 毕竟是官方强推 （除了某些时候会有些麻烦

- hexo 没用过

- hugo 方便，简洁，就暂时先用这个了


# How to install hugo ?

刚安装好的 ubuntu 直接 apt-get 可能没有 hugo 的包，只需 update 即可  

```shell
sudo apt-get update

sudo apt-get install hugo
```
